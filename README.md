# SherryScript
<p>
Features: <br>
	unwrapping numbers, evaluations, strings. <br>
	Declaring variables. <br>
	Cannot unwrap variables. <br>
To add a new function. <br>
1. Add a new elif statement in the Lexer <br>
2. Add a new if statement to parse it <br>
</p>
#Code examples
```
$var = 1
if $var == 1 then
	unwrap "True"
endif
unwrap "Hello world"
```
